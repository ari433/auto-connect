# public/ — static assets

## Brand logo

Put your AUTO CONNECT logo here as **`logo.png`** (this exact filename):

```
public/logo.png
```

It appears automatically in the header, footer and admin panel. If the file is
missing, the site falls back to the built-in text wordmark, so nothing breaks.

Tips:
- A transparent PNG works best. Height is scaled to ~36px; width auto.
- If the logo looks bad on the dark footer, provide a light version and tell us
  — we can wire a separate `logo-light.png`.
